## 🚀 Quick Start

### **Download and Run**
1. **Download**: `RentalGuideMY_ShareWithFriends.zip`
2. **Extract** the zip file
3. **Run**: `start_all.bat` (Windows) or `docker-compose up -d`
4. **Access**: http://localhost:3000

### **Prerequisites**
- Python 3.11+
- Node.js 18+
- Docker (optional)
